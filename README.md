# CNRoutingAssignment

This repository contains the Cisco Packet Tracer files developed for the Computer Networks routing assignment. The tasks include basic router configuration, static routing, and dynamic routing using RIP and EIGRP protocols.

## Repository Contents

### Basic Router Configuration.pkt

Basic router configuration:

* Hostname configuration
* Console password setup
* Console login authentication

### Dynamic Routing Configuration.pkt

Static routing implementation:

* IP address configuration
* Static route setup
* Network connectivity testing

### Static Routing Configuration.pkt

RIP routing implementation:

* RIP protocol configuration
* Connectivity verification between all networks

EIGRP routing implementation:

* EIGRP protocol configuration
* Connectivity verification between all networks

## Testing and Verification

All configurations were tested to ensure successful communication between the connected networks. Verification was performed using:

* Ping tests between devices and networks
* Routing table verification using router commands
* Connectivity checks across all configured networks
